# Classic Dental Scheduling System

A full-stack, production-ready web-based scheduling system for Classic Dental Clinic.

## Tech Stack
- **Frontend:** React (JSX), CSS
- **Backend:** Node.js, Express, JWT Auth
- **Database:** (Recommended: PostgreSQL or MongoDB)

## Features
- Responsive, sea green/white/light gray theme
- Secure authentication & role-based access
- Appointments, calendar, patients, dentists, reports, settings
- Prevents double-booking, detects conflicts
- Clean, maintainable code structure

## Getting Started

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm start`

### Backend
1. `cd backend`
2. `npm install`
3. `npm run dev`
